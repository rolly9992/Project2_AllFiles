My name is Rob and I've worked in Data Analytics for over 10 years. What is interesting about working in data is that there is always plenty to learn. Effectively one has to commit to being a lifelong student.

This was my first attempt at building a website. HTML, CSS, Plotly, Bootstrap and Flask were all new to me, despite using python for nearly a decade. There is quite a bit involved. In a way, it felt like learning chess for the first time, trying to get everything to work together as a team. And frankly, there is room for improvement. But, you have to start somewhere!

After doing this project I see that this is only a starting point, not the end. In addition, now I see how much of an important challenge this is, trying to filter text to the proper emergency channels.
